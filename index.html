ホーム

このサイト全体の入り口です。植栽一覧・サイトマップ・記録の閲覧と編集へ進めます。

記録一覧

表示

選んだ植栽が含まれる記録だけを表示します。記録日の並びは上の「並び順」で新しい順／古い順を選べます。エリアの絞り込みも使えます。

バックアップ

記録はサーバーに保存されています。必要なら一覧の内容を JSON ファイルでダウンロードできます（画像はリンクのまま含まれます）。

ホームについて

このページがサイトのトップ（index.html）です。ここから エリア一覧 と 植栽一覧 に進み、各ページの時系列や詳細をたどれます。

植栽時系列では、マスタや過去の記録にだけ出ている名前も含めて植栽を選び、その植栽が載っている記録を日付順に並べます。表示形式の選択はこのブラウザに記憶されます。

新しい記録の追加や編集・削除、植栽名マスタの変更は「記録を追加・編集」から行います。エリア写真の整理は各エリア時系列ページやエリア管理ページから行えます。

ブラウザのアドレスが http や https で始まる形で開いている必要があります。

リンク集のURLを設定してください

「リンク集へ戻る」は、入口のホームへのリンクです。インターネット上の本番サイトでは、次のいずれかで先のアドレスを指定してください。

  • data/hub-link.json の linkCollectionUrl に、リンク集のページのURL（https://… で始まるもの）を書く
  • または、HTML の先頭付近の window.__PLANTING_HUB_URL__ に同じURLを書く

自分のパソコン上のフォルダだけで見ているときは、フォルダの並びに合わせて自動で開くようになっています。
