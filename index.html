この一覧について

表示している植栽のリストは、サイト用のデータファイルから読み込んでいます。

データを書き換えたあとは、ファイル本体と、このページに埋め込んであるコピー用データの両方を同じ内容にしておくと、どこで開いてもずれません。

パソコンで HTML をダブルクリックして開いているだけのときは、ブラウザの仕組み上、ファイルではなく埋め込みのコピーが使われることがあります。ウェブサーバー経由（アドレスが http や https で始まる）で開くと、ファイルから最新を読み込めます。

リンク集のURLを設定してください

「リンク集へ戻る」は、入口のトップページへのリンクです。インターネット上の本番サイトでは、次のいずれかで先のアドレスを指定してください。

  • data/hub-link.json の linkCollectionUrl に、リンク集のページのURL（https://… で始まるもの）を書く
  • または、HTML の先頭付近の window.__PLANTING_HUB_URL__ に同じURLを書く

自分のパソコン上のフォルダだけで見ているときは、フォルダの並びに合わせて自動で開くようになっています。
