Growth log

成長記録

エリアと植栽を選び、写真とメモを Vercel（Blob + KV）に保存します。一覧のマスタは data/plants.json から読み込みます（HTML を直接開いたときは plants-embed を使います）。file:// のままでは API に届かないため、一覧取得・保存にはデプロイ URL または vercel dev が必要です。

API 接続

トークンはブラウザの localStorage にだけ入ります。Vercel の環境変数 GROWTH_UPLOAD_TOKEN を設定している場合は、同じ値を入力してください。未設定の API は誰でも読み書きできるため、公開 URL では必ずトークンを設定してください。
