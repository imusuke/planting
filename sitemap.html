サイトマップ

一覧・時系列・詳細・編集のつながりを、できるだけシンプルに確認できます。

ページ構成

  • ホーム root

    サイト全体の入口です。エリア一覧と植栽一覧から、時系列・詳細・編集へ進みます。

    • エリア一覧 browse

      エリアごとの入口です。各エリアの時系列へ進みます。

      • エリア一覧編集 edit

        一覧全体のエリア名やエリアIDを編集します。

      • エリア時系列 timeline

        各エリアの写真と記録を時系列で閲覧します。

        • エリア時系列編集 edit

          エリア写真やメモを編集します。

    • 植栽一覧 browse

      植栽ごとの入口です。植栽詳細と編集画面へ進みます。

      • 植栽一覧編集 edit

        一覧全体の植栽名を編集します。

      • 植栽詳細 detail

        その植栽の写真や説明を確認します。

        • 植栽記録編集 edit

          その植栽の記録や写真を追加・編集します。

        • 植栽詳細編集 edit

          植栽詳細の説明文や概要を編集します。

    • サイトマップ guide

      ページのつながりを確認するための案内ページです。
